Step 9. Label-free Prompt Proxy

Proxy construction does not use correctness labels. It only uses standardized observable features.

  • observability_score = 0.45*z(numeric_answer_rate) + 0.30*z(execution_success_rate) + 0.15*z(text_answer_extract_rate) + 0.10*z(execution_answer_recovery_rate)
  • consensus_score = 0.55*z(answer_consensus_margin) + 0.25*z(mean_rollout_score) - 0.20*z(answer_diversity_rate)
  • reward_proxy = tanh((0.60 * observability_score + 0.40 * consensus_score) / 2.0)
  • proxy_bucket = not_even_sampled-like if observability_score <= -0.039
  • proxy_bucket = majority_but_wrong-like if observability is above that cut but consensus_score <= 0.610
  • Otherwise the prompt is assigned to predicted_correct-like.
